# SC!32 Language

SC!32 is a custom HTML-like programming language extension for VS Code.

- Syntax highlighting for `<sc!32>`, `<str>`, `<print>` tags
- Autocomplete snippets for creating strings and print statements
- Supports `.sc32` file extension
- This was made by Me an iranian dude
- Python interpeter 
- ALL FIELS MUST BE IN PATH: `C:/sc32`
- And to use interpeter you must write `THE EXACT FILE NAME` or it wont find yur file to run (still working on)
- also interpreter.py must be in the path `C:/sc32`
